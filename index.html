Features

메일 작업의 모든 순간을 덜어주는 기능

⚡️

요약 & 번역

선택 영역이나 붙여넣기만으로 한국어/영어 요약과 번역을 분리된 버튼으로 실행.

🧭

감정 분석

가장 지배적인 감정을 한 줄 근거와 함께 반환. 신뢰도 수치는 제외.

✉️

회신 템플릿

요약 기반 영어 회신 템플릿 자동/수동 생성, 맞춤 지시 300자 제한, 무관 지시는 차단.

🖱️

컨텍스트 메뉴

우클릭 → 선택 영역 요약/번역(한국어) 실행, 팝업 자동 오픈.

🎨

테마 & 언어

라이트/다크 모드, UI 언어 ko/en 토글. 번역 버튼은 영어 UI에서는 숨김 처리.

💾

상태 저장

입력·요약·감정·번역·템플릿 결과를 로컬에 보존, 팝업 닫아도 유지.

Workflow

3단계로 끝나는 메일 처리

1

붙여넣기 혹은 선택

이메일을 2,000자까지 붙여넣거나, 페이지에서 영역을 선택 후 우클릭.

2

요약 · 번역 · 감정

버튼 하나씩 독립 실행. 눌러진 기능 결과만 표시되어 깔끔합니다.

3

회신 템플릿 복사

자동 생성 혹은 수동 생성 버튼으로 회신 초안을 바로 클립보드에.
