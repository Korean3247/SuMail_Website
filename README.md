# SuMail Website

SuMail 크롬 확장(요약·번역·감정 분석·회신 템플릿 생성)의 랜딩 페이지입니다. 단일 HTML/CSS/JS로 작성되어 누구나 바로 열어볼 수 있고, 한국어/영어를 토글할 수 있습니다.

## 주요 기능
- 히어로 섹션: 제품 태그라인, 주요 CTA(크롬 웹스토어), 핵심 지표(붙여넣기/선택 지원, 자동·수동 템플릿, 지원 언어).
- 미리보기 카드: 요약, 감정 분석, 회신 템플릿 UI를 카드 형태로 보여주는 데모 블럭.
- 기능/워크플로우: 6가지 기능 카드와 3단계 사용 흐름 설명.
- 다국어 토글: `ko/en` 스위치로 텍스트를 전환하며, `localStorage`에 선택 언어를 저장.
- 인터랙션: 스크롤 시 `IntersectionObserver`로 카드 페이드업 애니메이션, 새로고침 시 최상단으로 이동.

## 기술 스택
- 정적 사이트: `index.html`, `styles.css`, 간단한 인라인 스크립트만 사용.
- 디자인: CSS 커스텀 프로퍼티로 컬러/반경/쉐도우 관리, 그리드/플렉스 레이아웃과 라이트 톤 배경 그래디언트.
- 에셋: `assets/`에 로고 3종(PNG), `fonts/`는 커스텀 폰트 추가 시 사용 가능.

## 로컬에서 보기
1. 저장소 루트에서 바로 `index.html`을 브라우저로 열어도 됩니다.
2. 간단한 로컬 서버가 필요하다면:
   ```bash
   python3 -m http.server 4000
   # 브라우저에서 http://localhost:4000 접속
   ```

## 파일 구조
- `index.html` — 페이지 마크업과 i18n 스크립트.
- `styles.css` — 전체 레이아웃/타이포/애니메이션 스타일.
- `assets/` — SuMail 로고 이미지.
- `fonts/` — (옵션) 커스텀 폰트 배치용 디렉터리.

## 커스터마이징 가이드
- 텍스트/i18n: `index.html` 상단의 `I18N` 객체에서 한국어·영어 문구를 수정합니다. 새 언어를 추가하려면 `I18N`에 키를 추가한 뒤 `applyLang` 호출 시 사용하세요.
- 색상/스타일: `styles.css` 최상단의 `:root` CSS 변수(배경, 포인트 컬러 등)를 수정해 브랜딩에 맞출 수 있습니다.
- CTA 링크: 히어로와 CTA 섹션의 크롬 웹스토어 링크를 실제 배포 URL로 교체하세요.

## 배포
정적 사이트이므로 GitHub Pages, Netlify, Vercel과 같은 정적 호스팅 서비스에 그대로 업로드하면 됩니다. 별도의 빌드 단계나 서버 구성은 필요 없습니다.
